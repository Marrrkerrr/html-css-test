# Web Development Memorization Quiz

A static English quiz for memorizing HTML, CSS, Bootstrap, JavaScript, JSON, browser basics, and web development concepts.

## Features

- 141 questions
- Single-answer and multiple-answer questions
- Correct answers become green
- Wrong selected answers become red
- Score, mistakes, and accuracy counters
- Practice mistakes mode
- Progress saved in browser localStorage
- Works as a simple GitHub Pages website

## Files

- `index.html` — the quiz
- `.nojekyll` — disables Jekyll processing on GitHub Pages
- `README.md` — instructions

## How to publish on GitHub Pages

1. Create a new GitHub repository, for example `web-dev-quiz`.
2. Upload these files:
   - `index.html`
   - `.nojekyll`
   - `README.md`
3. Open the repository on GitHub.
4. Go to **Settings**.
5. Go to **Pages**.
6. In **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
7. Save.
8. GitHub will give you a public link like:

```text
https://your-username.github.io/web-dev-quiz/
```

Send that link to classmates.
